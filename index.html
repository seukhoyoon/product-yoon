문제

왜 대부분의 세일즈 팀은
제대로 싸우지 못하는가

01

보상 설계가 감에 의존한다

엑셀, 전임자 관행, 업계 평균 — 우리 팀에 맞는 구조가 아니다.

02

의무 활동은 있고, 책임은 없다

콜 횟수, 미팅 수를 정해놓고 미이행해도 인센티브는 그대로다.

03

SDR·AE·AM이 같은 룰로 움직인다

역할이 다르면 보상 구조도 달라야 한다. 지금은 그렇지 않다.

BASE CAMP는 이 세 가지를 한 번에 해결합니다.

솔루션

BASE CAMP가 하는 것

⚙

맞춤 보상 구조 자동 설계

조직 체계, 매출, 영업 방식, 난이도를 입력하면 SDR·AE·AM별 최적 인센티브 구조를 즉시 생성한다.

⚡

의무 미이행 시 자동 차감 규칙

활동 목표를 설정하고, 미달 시 인센티브가 자동으로 조정된다. 규칙이 아니라 시스템이 팀을 움직인다.

◈

전략적 평가 체계 통합

보상만이 아니다. 성과 평가, 페널티, 승진 기준까지 하나의 전략으로 연결된다.

컨설턴트 없이.  스프레드시트 없이.  몇 분 안에.

작동 방식

3단계로 완성되는 세일즈 보상 전략

①

입력

조직 구조, 매출 목표, 영업 방식, 직군별 역할, 난이도를 입력한다.

②

설계

BASE CAMP가 SDR·AE·AM별 인센티브 구조, 의무 활동 기준, 차감 규칙, 평가 체계를 자동으로 생성한다.

③

적용

완성된 전략을 팀에 바로 도입한다. 수정이 필요하면 파라미터만 바꾸면 된다.
